# FluxiON Web Brand Kit

Este repositorio ahora incluye una base de identidad visual lista para aplicar en web y piezas digitales.

## Archivos

- `brand-guidelines.md`: guia resumida de color, tipografia y estilo.
- `brand-tokens.css`: tokens CSS y clases base para interfaz.

## Uso rapido

1. Importa `brand-tokens.css` en tu proyecto.
2. Carga las fuentes Poppins e Inter.
3. Usa los tokens para mantener consistencia de marca en componentes.

## Reglas clave de marca

- Proporcion recomendada: 70% fondos claros, 20% azul, 10% naranja.
- Naranja para llamados a la accion y foco visual.
- Azul para estructura, encabezados y confianza visual.
